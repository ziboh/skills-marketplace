# Skills Marketplace

Auto-generated [marketplace.json](marketplace.json) parsed from [VoltAgent/awesome-agent-skills](https://github.com/VoltAgent/awesome-agent-skills).

每天自动同步，生成 Skill Hub 兼容的技能市场数据。

## Usage

```
marketplace.json
```

直接引用 GitHub Raw URL 即可作为 Skill Hub 的 `marketplace-json` 源。

## Workflow

`.github/workflows/sync-marketplace.yml` 每天 UTC 00:00 自动运行，也可手动触发。
